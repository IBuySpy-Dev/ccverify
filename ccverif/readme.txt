CCVerification Object



summary:
===============
Absolute verification of credit card numbers. Used before sending
a card out for debit to ensure that the card has the highest
probability of being a valid card.



documentation:
===============
CCVerification in a VBScript class and behaves like an object in vbscript.
CCVerification exposes various properties and methods that can be used
to determine if an entered credit card number can be verified using common
well-known algorithms. Please view ccverification_class.inc for complete
documentation on available properties and methods of the object.



installation:
===============
1.) you will need the CCVerification_class.inc file and the 
    ccverification_class.asp file. The INC file contains the
    definition of the object and the accompanying reference
    for the object. The ASP file contains the call to the object.


2.) place the required files from step 1 into the root directory 
    on the server.


3.) navigate to ccverification_class.asp on your website with your web browser 
    to test.
